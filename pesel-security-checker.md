- **What it does**: A Python script generates valid PESEL numbers for a specific age range (e.g., 40-50) and full age range (16-100), then uses JohnTheRipper on Kali Linux to crack a ZIP file password set as a PESEL.
- **Key Insight**: PESEL-based passwords lack robust security and are vulnerable to dictionary attacks due to their predictable structure.  
- **Tools**: Python, JohnTheRipper, Kali Linux.
